OpenShift Virtualization GitOPs Repository
------------------------------------------
This repository is used to manage the OpenShift Virtualization GitOps Deployments in a Gitops Manner. 

Requirements
------------
* RHEL 8.x tested on RHEL 8.6

Quick start
------------
```
./scripts/install.sh
```
